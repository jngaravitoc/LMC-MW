The dynamical friction equation is modified in such a way that:

\begin{equation}
a_{df} = -4\pi G^2 M_{sat} \rho \alpha Ln(\Lambda)  \left( erf(X) - \dfrac{2X}{\sqrt{\pi}}exp(-X^2)   \right) \vec{v}/v^3
\end{equation}
